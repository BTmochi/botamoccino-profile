# botamoccino-profile

ぼたもちーのさんのプロフィールサイトです。VRChatでの活動、コミュニティ、各種リンク、ファンアートをまとめています。

## 構成

- `index.html` - ページ本体、プロフィール情報、リンク、ライトボックスのHTML
- `style.css` - レイアウト、配色、カード、アニメーション、ライトボックスのスタイル
- `script.js` - ファンアート一覧の生成、ライトボックス表示、スクロール時アニメーション
- `img/` - ファンアート画像やOGP画像などの画像ファイル

## ローカルで確認する方法

HTML/CSS/JavaScriptだけで動く静的サイトです。`index.html` をブラウザで直接開いて確認できます。

ローカルサーバーで確認する場合は、プロジェクトフォルダで次を実行します。

```bash
python -m http.server 8765 --bind 127.0.0.1
```

その後、ブラウザで次を開きます。

```text
http://127.0.0.1:8765/
```

## ファンアートの追加・編集

ファンアートの一覧は `script.js` の `fanartData` で管理しています。

```js
{
  img: "img/fanart-01.jpg",
  author: "作者名",
  handle: "@user_id",
  date: "",
  caption: "キャプション",
  url: "元ポストのURL"
}
```

画像を追加する場合は、`img/` に画像ファイルを置き、`fanartData` に1件追加してください。

ユーザーアイコンは `handle` をもとに自動取得されます。個別のアイコン画像を指定したい場合は、データに `avatar` を追加できます。

```js
{
  img: "img/fanart-11.jpg",
  author: "作者名",
  handle: "@user_id",
  avatar: "img/avatar-user-id.jpg",
  date: "",
  caption: "キャプション",
  url: "元ポストのURL"
}
```

## 主な機能

- プロフィール・経歴・コミュニティ・リンクの表示
- ファンアートのグリッド表示
- ファンアートのライトボックス表示
- ライトボックス内で画像全体を表示
- 作者アイコンと作者名の表示
- スクロール時の浮き上がりアニメーション

## メモ

外部リソースとして、Google Fonts、各サービスのfavicon、ユーザーアイコン取得用の `unavatar.io` を利用しています。
